# DarwiNN
Distributed neuroevolution framework based on Pytorch.

## Dependencies

DarwiNN depends on:
* Python (3.6+)
* MPI
* Pytorch (1.1.0+)
* DEAP (for specific black box optimization uses)

## Installation

DarwiNN is pip-installable with the following commands:

```
python setup.py bdist_wheel
pip install dist/*.whl
```
